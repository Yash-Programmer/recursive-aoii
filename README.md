# Cascading Semantic Age — Reproducibility Artifact

Reproducibility code for the paper

> **Recursive AoII in Multi-Hop Wireless Sensor Cascades.**

Every closed form in the paper is verified **two independent ways** — symbolically (exact, via SymPy) and empirically (Monte-Carlo with a 95% confidence interval) — and the entire numerical study, including every figure and reported number, regenerates from a **single command**.

## Quick start

```bash
python -m pip install -r requirements.txt

# full: symbolic checks + 42 tests + experiments E1–E7 + figures  (~8–10 min)
python code/reproduce.py

# fast: skip the 10^6-event sweep (symbolic checks + tests + figures)
python code/reproduce.py --fast
```

A successful run prints `8/8` symbolic identities `[PASS]`, `42 passed` for the test
suite, regenerates `experiments/E*_data.json`, and writes the seven paper figures to
`figures/plots/`. All randomness is seeded, so results are deterministic across machines.

Tested with **Python 3.10**. Dependencies: `numpy`, `scipy`, `sympy`, `numba`, `matplotlib`.

## What gets checked

`code/reproduce.py` runs four stages; each fails loudly (`assert`) if anything is off.

| Stage | What it asserts |
|---|---|
| **1. Symbolic** | 8 exact SymPy identities: the residual integral `E[φ(A_k)] = ν/(μ_k+2ν)`, the recursion→product identity, the threshold fixed point, and the `K=1` mean-AoII closed form (which recovers the single-hop AoII of Maatouk et al.). |
| **2. Tests** | 42 `pytest` cases: JIT-vs-reference engine match (bit-identical on shared randomness), Lemma 1 parity enumeration, the product formula `p_K`, threshold policies, the N-state symmetric form, erasure-channel substitution, the exact joint-CTMC covariance / NQD checks, the exact joint benchmark over per-link threshold vectors (always-push vs `joint.py`; the calibration gap), and the unrestricted (genie) optimum (always-push exact, saturation bound, genie below the threshold class, error-cancellation withholding). |
| **3. Experiments** | E1 product-formula sweep (K≤12), E2 parity identity, E3 budget reallocation at scale (K=6), E4 policy comparison, E5 heavy-tail + tree path-product, **E6 exact joint benchmark** (Table II: always-push, surrogate- and achieved-calibrated decoupled policy, threshold-class optimum, the two gaps), **E7 unrestricted (genie) optimum** (price of decentralization: genie gap at K=2, depth scaling, error-cancellation mechanism) → `experiments/E*_data.json`. |
| **4. Figures** | Regenerates F1–F6 to `figures/plots/`. |

### Headline results reproduced from first principles
- **Product formula:** `p_K = ½ [ 1 − ∏_k μ_k/(μ_k + 2ν) ]`
- **Mean Recursive AoII:** `E[Δ^R_K] = − π_Mᵀ A⁻¹ 1` (exact at every `K`; elementary at `K = 1, 2, 3`)
- **Universal age bound:** `E[Δ^R_K] ≤ p_K/ν` for every policy (`< 1/(2ν)` under always-push; never-transmit attains `1/(2ν)` exactly) — semantic age **saturates** with hop count
- **Symmetric N-state:** `p_K = (1 − 1/N) [ 1 − ∏_k μ_k/(μ_k + Nν/(N−1)) ]`
- **Decoupling gap is governed by calibration (exact joint benchmark, E6 / Table II):** against the optimum over all per-link threshold vectors, the decentralized policy of Algorithm 1 with the offline *surrogate* calibration sits 30–53% above the optimum, while the *achieved* (measured-rate) calibration is within 1.8% at every operating point. The per-link threshold structure is the right one; the measured-rate adjustment is what makes it near-optimal. The gap is bounded throughout by the proven `G_K ≤ 2R`; the unrestricted (genie) optimum, which augments the state with the monitor's end-to-end wrong-run age, is computed exactly in **E7** (below).
- **Near-optimal at scale (E3, K=6):** beyond the exactly solvable lengths, no reallocation of the per-hop thresholds at fixed total budget beats the achieved-rate-calibrated policy by more than a few percent in mean Recursive AoII, and the entire residual gain comes from lowering the *first* relay's threshold to spend slack budget on the bottleneck link facing the fast source — budget *allocation*, not the decentralized threshold rule (cf. Corollary on bottlenecks).
- **Price of decentralization (exact genie optimum, E7):** the per-link threshold rule is the optimal *decentralized* structure, but the unrestricted optimum — a controller that may condition on the global mismatch, in particular the monitor's end-to-end wrong-run age that no relay can observe — lies **21–35% below it at K=2** (31.2%, 21.1%, 35.2% across the three Table II operating points), and the gap **grows with depth** (≈31→42→50% for K=2→4). The entire gap is realized by *withholding* a forward that would propagate an upstream error onto a monitor made correct by **error-cancellation** (state `M=(1,1)`: the genie transmits there 0% of the time vs 95% when the forward actually repairs the monitor). Detecting that state requires the source, so the gap is a **fundamental price of decentralization**, not a deficiency of the rule. Strikingly, the genie holds R-AoII *depth-independent* under the per-link budget (≈0.17 ∀K) — a budgeted strengthening of the saturation bound — while always-push and the threshold class climb toward `1/(2ν)`.

The extra developer probes in `code/verify/` print the precise numbers cited in the
theory sections, including the Tier-2 lemmas (universal age bound, negative quadrant
dependence, causal conditional independence). Run e.g. `python code/verify/verify_strongaccept.py`.

## Verification protocol (as in the paper's appendix)

A closed form is accepted only when **(a)** its symbolic re-derivation is exact **and**
**(b)** the Monte-Carlo 95% confidence interval brackets it. Simulations use 10⁶ events ×
20 fixed seeds (`0`–`19`); the JIT kernel and the pure-python reference consume identical
pre-drawn randomness and therefore produce bit-identical output, which the engine-match
test guards. The simulator's age accounting is itself validated by the deterministic parity
identity of Lemma 1, which holds at every recorded event.
