# Cascading Semantic Age вҖ” Reproducibility Artifact

Reproducibility code for the paper

> **Cascading Semantic Age: Recursive Age of Incorrect Information (R-AoII) over Multi-Hop Wireless Sensor Networks.**

Every closed form in the paper is verified **two independent ways** вҖ” symbolically (exact, via SymPy) and empirically (Monte-Carlo with a 95% confidence interval) вҖ” and the entire numerical study, including every figure and reported number, regenerates from a **single command**.

## Quick start

```bash
python -m pip install -r requirements.txt

# full: symbolic checks + 36 tests + experiments E1вҖ“E5 + figures  (~2вҖ“3 min)
python code/reproduce.py

# fast: skip the 10^6-event sweep (symbolic checks + tests + figures)
python code/reproduce.py --fast
```

A successful run prints `8/8` symbolic identities `[PASS]`, `36 passed` for the test
suite, regenerates `experiments/E*_data.json`, and writes the seven paper figures to
`figures/plots/`. All randomness is seeded, so results are deterministic across machines.

Tested with **Python 3.10**. Dependencies: `numpy`, `scipy`, `sympy`, `numba`, `matplotlib`.

## What gets checked

`code/reproduce.py` runs four stages; each fails loudly (`assert`) if anything is off.

| Stage | What it asserts |
|---|---|
| **1. Symbolic** | 8 exact SymPy identities: the residual integral `E[ПҶ(A_k)] = ОҪ/(Ој_k+2ОҪ)`, the recursionвҶ’product identity, the threshold fixed point, and the `K=1` mean-AoII closed form (which recovers the single-hop AoII of Maatouk et al.). |
| **2. Tests** | 36 `pytest` cases: JIT-vs-reference engine match (bit-identical on shared randomness), Lemma 1 parity enumeration, the product formula `p_K`, threshold policies, the N-state symmetric form, erasure-channel substitution, and the exact joint-CTMC covariance / NQD checks. |
| **3. Experiments** | E1 product-formula sweep (KвүӨ12), E2 parity identity, E3 decoupling gap, E4 policy comparison, E5 heavy-tail + tree path-product вҶ’ `experiments/E*_data.json`. |
| **4. Figures** | Regenerates F1вҖ“F6 to `figures/plots/`. |

### Headline results reproduced from first principles
- **Product formula:** `p_K = ВҪ [ 1 вҲ’ вҲҸ_k Ој_k/(Ој_k + 2ОҪ) ]`
- **Mean Recursive AoII:** `E[О”^R_K] = вҲ’ ПҖ_MбөҖ AвҒ»В№ 1` (exact at every `K`; elementary at `K = 1, 2, 3`)
- **Universal age bound:** `E[О”^R_K] вүӨ p_K/ОҪ < 1/(2ОҪ)` вҖ” semantic age **saturates** with hop count
- **Symmetric N-state:** `p_K = (1 вҲ’ 1/N) [ 1 вҲ’ вҲҸ_k Ој_k/(Ој_k + NОҪ/(NвҲ’1)) ]`

The extra developer probes in `code/verify/` print the precise numbers cited in the
theory sections, including the Tier-2 lemmas (universal age bound, negative quadrant
dependence, causal conditional independence). Run e.g. `python code/verify/verify_strongaccept.py`.

## Verification protocol (as in the paper's appendix)

A closed form is accepted only when **(a)** its symbolic re-derivation is exact **and**
**(b)** the Monte-Carlo 95% confidence interval brackets it. Simulations use 10вҒ¶ events Г—
20 fixed seeds (`0`вҖ“`19`); the JIT kernel and the pure-python reference consume identical
pre-drawn randomness and therefore produce bit-identical output, which the engine-match
test guards. The simulator's age accounting is itself validated by the deterministic parity
identity of Lemma 1, which holds at every recorded event.
